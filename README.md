# Egypt Real Estate Market Analysis

## Project Overview
This project analyzes the real estate market in Egypt using data scraped from Bayut.

## Tools Used
- Python
- Pandas
- BeautifulSoup
- Tableau
- Data Visualization

## Project Steps
1. Web scraping real estate data
2. Data cleaning and preprocessing
3. Feature engineering (price per square meter)
4. Exploratory data analysis
5. Building an interactive dashboard

## Key Insights
- North Coast has the highest price per square meter.
- Apartments are the most common property type.
- New Cairo has the highest number of listings.
- Property price increases with area.
